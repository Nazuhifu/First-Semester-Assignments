# RESPONSI 1 PRAKTIKUM STRUKTUR DATA DAN ALGORITMA

Buatlah sebuah program (bebas memakai tema apapun) menggunakan bahasa C++ sekaligus melakukan analisis kompleksitas dengan memanfaatkan minimal 15 poin struktur data dengan rincian sebagai berikut:

- List : 5 poin
- Stack : 5 poin
- Queue : 5 poin
- Set : 5 poin
- Map : 5 poin
- Tree : 15 poin
- Binary Tree : 10 poin
Total keseluruhan poin : 50 poin

Catatan: poin dihitung berdasarkan jenis struktur data, bukan banyaknya struktur data. Dengan  demikian, apabila program memiliki list sebanyak 3, maka tetap dihitung 5 poin bukan 15 poin.

Kumpulkan dalam bentuk 1 file .zip yang berisi:
1. File source code dalam bahasa C++ (*.cpp)
2. File laporan dalam bentuk PDF (*.pdf)

Format: NIM_Nama_Responsi1.zip
Contoh: L0123456_Ahmad Segfault_Responsi1.zip
